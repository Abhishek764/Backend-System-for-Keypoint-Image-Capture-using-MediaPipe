# Environment Variables

Copy this file to `.env` in the backend directory and fill in your values.

## Server Configuration
PORT=3000
NODE_ENV=development
TZ=UTC

## PostgreSQL Database Configuration
POSTGRES_HOST=localhost
POSTGRES_PORT=5432
POSTGRES_USER=postgres
POSTGRES_PASSWORD=your_postgres_password
POSTGRES_DB=keypoints_db
PG_DUMP_PATH=pg_dump

## MongoDB Configuration
MONGODB_URI=mongodb://localhost:27017/images_db

## SendGrid Email Configuration
SENDGRID_API_KEY=your_sendgrid_api_key
SENDGRID_FROM_EMAIL=noreply@example.com
SENDGRID_TO_EMAIL=admin@example.com

## Backup Configuration
BACKUP_RETENTION_DAYS=7

## Python/MediaPipe Configuration
PYTHON_SCRIPT_PATH=../mediapipe/extract_keypoints.py
